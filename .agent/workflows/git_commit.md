---
description: Commit changes to Git
---

1. Check the status of the repository
   - run: `git status`

2. Stage all changes
   - run: `git add .`

3. Commit changes with a descriptive message
   - run: `git commit -m "update: progress on field resonance and battle ui"`
   - **User Action**: You can modify the message above to be more specific if you like.

4. Verify the commit
   - run: `git log -1`
